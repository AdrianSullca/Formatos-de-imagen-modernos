La diferencia que hay entre la imagen con tamaño real y la imagen con los atributos width y height, es el tamaño a que se redimensiona la imagen.
Cuando especificamos el width o heigh el navegador redimensiona la imagen real a los valores que le damos, si indicamos un width y height a una etiqueta de img y la imagen no llega a deformarse ya que obvia el height y este es calculado automaticamente por el navegador para no deformar la imagen, si quisieramos forzar el height manual pues se tendria que usar un fichero .css enlazado al html
